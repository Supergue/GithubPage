## 📋 Detaillierte Kurzanleitung

### **Schritt 1: GitHub Konto**
- Gehen Sie zu [github.com](https://github.com)
- **Sign up** klicken
- E-Mail und Passwort eingeben

### **Schritt 2: Repository erstellen**
- Oben rechts: **+** → **New repository**
- Name: `klopf-it-solutions`
- **Public** wählen
- **Create repository**

### **Schritt 3: Dateien hochladen**
- **Add file** → **Upload files**
- Alle Dateien und Ordner aus `github-pages/` rein ziehen:
  ```
  index.html, about.html, services.html, contact.html
  css/ (mit style.css)
  assets/ (mit Logos)
  _config.yml, README.md, etc.
  ```
- **Commit changes**

### **Schritt 4: GitHub Pages einschalten**
- **Settings** → **Pages**
- **Source**: `Deploy from a branch`
- **Branch**: `main`
- **Folder**: `/root`
- **Save**

### **Schritt 5: Website abrufen**
Nach 1–2 Minuten sehen Sie unter **Settings** → **Pages** die Live-URL:
```
https://YOUR_USERNAME.github.io/klopf-it-solutions
```

## 🎨 Anpassungen (Schnell gemacht)

### **Farbe ändern (Blau → andere Farbe)**
1. Datei `css/style.css` öffnen (auf GitHub)
2. Suchen: `#0b5fff` (blau)
3. Ersetzen mit Ihrer Farbe (z. B. `#ff6600` für orange)
4. **Commit**

### **Text/Inhalt ändern**
1. HTML-Datei öffnen (z. B. `index.html`)
2. Pencil-Icon klicken ✏️
3. Text bearbeiten
4. **Commit changes**

### **Kontakt-Info eingeben**
1. `contact.html` öffnen
2. Diese Zeilen suchen und ersetzen:
   - `kontakt@klopf-it.de` → Ihre E-Mail
   - `+49 (0) XXX` → Ihre Telefonnummer
   - Adresse + PLZ

### **Logo tauschen**
1. `assets/` folder öffnen
2. Alte Logodatei löschen
3. Neue Datei hochladen (GLEICHER NAME!)

## ❓ FAQ

**Wie lange dauert es?**  
Beim Hochladen: wenige Sekunden  
Beim Deployen: 1–2 Minuten (später: 30 Sekunden)

**Kann ich die Website offline anschauen?**  
Ja! Öffnen Sie `index.html` direkt im Browser.

**Das Kontaktformular funktioniert nicht?**  
Das ist normal – Sie müssen einen Service wie [formspree.io](https://formspree.io) verwenden. Siehe `README.md` → "Kontaktformular aktivieren"

**Kann ich Custom Domain nutzen?**  
Ja! Settings → Pages → Custom domain  
(Brauchen Sie eine Domain wie `www.klopf-it.de`)

**Wer kann die Website sehen?**  
Jeder mit der URL – es gibt keine Zugriffsbeschränkung.
